ข้อกำหนดในการใช้ WashiBot

อัปเดตล่าสุด:

เมื่อคุณใช้ WashiBot บน Discord หรือเว็บไซต์ของเรา คุณยอมรับข้อกำหนดเหล่านี้...

1) การยอมรับข้อกำหนด

การใช้งานบอทหรือเว็บไซต์ถือว่าคุณยอมรับเงื่อนไขทั้งหมด หากไม่เห็นด้วย โปรดหยุดใช้บริการทันที

2) การเปลี่ยนแปลงข้อกำหนด

เราอาจปรับปรุงข้อกำหนดเป็นครั้งคราว โดยจะแจ้งผ่านหน้าเว็บนี้ การใช้งานต่อไปหมายถึงคุณยอมรับข้อกำหนดที่แก้ไข

3) คุณสมบัติผู้ใช้งาน

คุณรับรองว่ามีอายุอย่างน้อย 18 ปี หรือได้รับความยินยอมจากผู้ปกครอง และมีความสามารถตามกฎหมายในพื้นที่ของคุณ

4) บัญชีผู้ใช้

  • คุณต้องใช้บัญชี Discord จริงและให้ข้อมูลที่ถูกต้อง
  • รับผิดชอบต่อคำสั่งและข้อมูลที่ส่งผ่านบอทหรือเว็บไซต์
  • เราอาจระงับบัญชีที่ละเมิดข้อกำหนด

5) การใช้งานที่ยอมรับได้

คุณตกลงว่าจะไม่:

  • ละเมิดกฎหมาย ลิขสิทธิ์ หรือสิทธิของผู้อื่น
  • ส่งเนื้อหาที่ผิดกฎหมาย อันตราย หรือมีมัลแวร์
  • พยายามเจาะระบบหรือรบกวนการทำงานของบอท/เว็บไซต์

นโยบายความเป็นส่วนตัว

อัปเดตล่าสุด:

WashiBot และเว็บไซต์ให้ความสำคัญกับความเป็นส่วนตัวของผู้ใช้...

1) ข้อมูลที่เก็บ

  • ข้อมูลบัญชี Discord: User ID, Username
  • ข้อมูลการใช้งาน: คำสั่ง, เวลาใช้งาน, การเข้าชมหน้าเว็บ
  • ข้อมูลเทคนิค: IP address, browser, device
  • คุกกี้: การตั้งค่าและตัวระบุไม่ระบุตัวตน

2) การใช้ข้อมูล

  • ปรับปรุงคำสั่งและบริการของบอท/เว็บไซต์
  • วิเคราะห์เพื่อปรับแต่งประสบการณ์ผู้ใช้
  • สื่อสารกับผู้ใช้ผ่านการแจ้งเตือน

3) การเปิดเผย/แชร์ข้อมูล

เราไม่ขายข้อมูล แต่แชร์กับผู้ให้บริการที่จำเป็นหรือตามกฎหมาย
